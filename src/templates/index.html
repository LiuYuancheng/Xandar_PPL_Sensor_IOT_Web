Welcome to Access the XAKA[Xandar Kardian] People Counting Radar IoT


This project will create a simple web server to provide the history chart to show the Xandar Kardian People counting sensor data. So when the engineer are installing the sensor with its IOT gateway. They don't need to use the computer to connect to the sensor directly to check the sensor state. They can login in the web sever running in the IOT gateway to finish the job.

Steps to Config the IoT Device
•   Allocate the XAKA sensors to the monitor area position as shown on the right side
•   Connect the Raspberry PI to the network router by wire / wifi.
•   Find the Raspberry PI IP address and log in the IoT Page with http://:5000
•   After login check the sensor info and change the config to link the IoT to your hub.
